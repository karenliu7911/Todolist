### Todolist 專案

- 使用技術
    - Django 6.0

- 安裝套件
    - pip install django


### 相關指令

- 啟動指令
     - django-admin startproject core .

- 啟動Server
    - python manage.py runserver

- 同步資料表
    - python manage.py makemigration
    - python manage.py migrate

- 建立管理員
    - python manage.py createsuperuser

- 新增APP
    - python manage.py startapp users
